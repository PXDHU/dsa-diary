# 680. Valid Palindrome II

**Link:** [LeetCode Problem - 680. Valid Palindrome II](https://leetcode.com/problems/valid-palindrome-ii/description/)

---

## Problem Statement

Given a string s, return true if the s can be palindrome after deleting at most one character from it.
---

## Example 1

**Input:**
s = "aba"

**Output:**
true

---

## Example 2

**Input:**
s = "abca"

**Output:**
true

---

## Constraints

 - `1 <= s.length <= 105`
 - `s consists of lowercase English letters.`
