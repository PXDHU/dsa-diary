# 217. Contains Duplicate

**Link:** [LeetCode Problem - 217. Contains Duplicate](https://leetcode.com/problems/contains-duplicate/description/)

---

## Problem Statement

Given an integer array nums, return true if any value appears at least twice in the array, and return false if every element is distinct.

---

## Example 1

**Input:**
nums = [1,2,3,1]

**Output:**
true

**Explanation:**
ans = [nums[0], nums[1], nums[2], nums[0], nums[1], nums[2]]
ans = [1,2,1,1,2,1]

---

## Example 2

**Input:**
nums = [1,2,3,4]

**Output:**
false

**Explanation:**
All elements are distinct.

---

## Constraints

- `1 <= nums.length <= 105`
- `-109 <= nums[i] <= 109`
